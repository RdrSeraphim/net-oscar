# Net::OSCAR
*alternatively: srp tries to "maintain" a 20-year-old perl library*

Implementation of AOL's OSCAR protocol for instant messaging (for interacting with AIM a.k.a. AOL IM a.k.a. AOL Instant Messenger - and ICQ, too!)

**Warning**: This is not the continuation of the module in the fullest sense. I'm mostly fiddling with making bots and patching this as necessary. I'm happy to take issues and PRs.
